# Program Overview

## Full Stack AWS Data Engineering Course

### Program Objective

This comprehensive program is designed to transform you into a proficient full-stack data engineer capable of designing, building, and maintaining end-to-end data pipelines on AWS. You will gain hands-on experience with modern data engineering tools and practices used in industry.

## Target Audience

- Software developers transitioning to data engineering
- Data analysts looking to move into data engineering roles
- Students interested in cloud-based data engineering
- Professionals seeking AWS data engineering skills
- Anyone aiming for roles like Data Engineer, Cloud Data Engineer, or ETL Developer

## Program Highlights

### 1. **Comprehensive Curriculum**
- 17 major modules covering all aspects of data engineering
- From foundational concepts to advanced distributed systems
- Aligned with industry best practices and current market demands

### 2. **Hands-On Learning**
- 11 capstone projects integrated throughout the course
- Real-world use cases from healthcare, retail, banking, and tech industries
- Live coding sessions and practical implementations

### 3. **AWS-Focused**
- Deep dive into AWS data services
- Cost-effective solutions using AWS free tier
- Best practices for cloud-based data engineering

### 4. **Latest Technologies**
- Apache Spark for big data processing
- Kafka for real-time streaming
- Terraform for infrastructure as code
- Jenkins for CI/CD pipelines
- Kubernetes for container orchestration

### 5. **Interview Preparation**
- Mock interview sessions
- Real interview questions from top companies
- System design problems for data engineers
- Behavioral and technical interview prep

## Course Structure

### Module Breakdown

| Module | Duration | Focus | Practical Work |
|--------|----------|-------|-----------------|
| 01 - Python | 3-4 weeks | Core programming, data structures, OOP | 4 mini projects |
| 02 - SQL | 3 weeks | Advanced queries, optimization, window functions | 5 SQL projects |
| 03 - Linux & Git | 2 weeks | Command line, version control, scripting | 2 assignments |
| 04 - AWS Cloud | 2 weeks | Account setup, IAM, S3, EC2, networking | Hands-on labs |
| 05 - AWS DE Services | 3 weeks | Lambda, Glue, Athena, Redshift, DynamoDB | 3 mini projects |
| 06 - Spark & PySpark | 3 weeks | RDD, DataFrames, Spark SQL, performance | 3 projects |
| 07 - Kafka | 2 weeks | Topics, producers, consumers, streaming | 1 project |
| 08 - Airflow | 2 weeks | DAGs, operators, scheduling, monitoring | 1 project |
| 09 - Terraform | 2 weeks | Infrastructure as Code, modules, workspaces | 1 project |
| 10 - Jenkins | 2 weeks | CI/CD, pipelines, deployments | 1 project |
| 11 - Docker & K8s | 2 weeks | Containerization, orchestration | 1 project |
| 12 - Data Warehouse | 1-2 weeks | Schema design, dimensional modeling, SCD | Modeling exercises |
| 13 - GenAI | 1-2 weeks | LLMs, prompt engineering, AI tools | 1 project |
| 14 - Real-time Projects | 2 weeks | End-to-end implementation | 4 projects |
| 15 - Capstone | 2-3 weeks | Integration project covering all topics | 1 major project |

**Total Duration**: 24-26 weeks (Full-time equivalent)

### Learning Methods
