## Program Duration

- **Total Duration**: 24-26 weeks (approximately 6 months)
- **Weekly Commitment**: 40-50 hours per week
- **Modules**: 17 major modules
- **Projects**: 11 capstone projects

## Learning Outcomes

By the end of this course, you will be able to:

1. Write production-grade Python code for data engineering
2. Write complex SQL queries and optimize them
3. Work with Apache Spark and PySpark
4. Design and implement ETL pipelines
5. Use AWS services for data engineering (Lambda, Glue, Athena, Redshift)
6. Implement real-time streaming with Kafka
7. Orchestrate workflows with Apache Airflow
8. Provision infrastructure with Terraform
9. Implement CI/CD pipelines with Jenkins
10. Containerize applications with Docker and Kubernetes
11. Design data warehouses and implement data models
12. Work with modern GenAI tools for data engineering

## Prerequisites

- Basic programming knowledge
- Understanding of databases and SQL (basics)
- Familiarity with command line
- An AWS account (free tier eligible)
- Computer with 8GB+ RAM

## Course Structure

### Phase 1: Foundations (Weeks 1-4)
- Python Basics
- SQL Fundamentals
- Linux & Git
- AWS Cloud Foundation

### Phase 2: Core Data Engineering (Weeks 5-12)
- Advanced Python for Data Engineering
- Advanced SQL
- Apache Spark & PySpark
- AWS Data Services

### Phase 3: Advanced Topics (Weeks 13-20)
- Kafka & Streaming
- Airflow Orchestration
- Terraform & Infrastructure as Code
- Jenkins & CI/CD

### Phase 4: Integration & Capstone (Weeks 21-26)
- Docker & Kubernetes
- Data Warehouse Modeling
- GenAI for Data Engineers
- Capstone Projects
